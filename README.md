# geoffprice.vercel.app

Personal site — notes on AI leadership. Each entry is one durable idea from a book, paired
with a prompt you can run on your own AI to apply it to a real decision.

Next.js (App Router), deployed on Vercel. Pushes to `main` deploy automatically.

The decision skills behind the essays live at
https://github.com/geoff-price/ai-leadership-playbook
